Glossary

A collection of commonly used BIM phrases and acronyms, explained and searchable.
Click on a Term for more information.

Term

Description

3D Modelling

Creating a 3D model using a computer. This is a 3D representation of a drawing.

The user is able to move around and in and out of the model exploring its 3D geometry.

Benefits

A positive outcome resulting in time or money saved or an increase in safety

We talk about benefits, what we're saying here is a positive or better alternative to the previous method of work.

BEP

Building Information Modelling Execution Plan

A BEP, a document that outlines and defines which BIM tools will be used on a project. It will also include general information about the project, how the BIM tools will be used on a project and explain each project role in relation to the information captured.

BIM

Building Information Modelling

A digital way of capturing, analysing and storing data within any type of construction project. This usually includes attaching this information to a model. BIM does not refer to a model but a theory or method of working.

CDE

Common Data Enviroment

An online digital storage centre, similar to the G drive accept we are able to give access to the people that need it. This enviroment is used to share all of our project data, only those who need access are granted access to it. Everything from documents, photos and drawings, to models and pointclouds.

Clash Detection

Comparing two models and finding design conflicts.

Using 3D Modelling software to compare two or more models in the same 3D enviroment. This can highlight where they clash in the real world without risking any damage. For example we can simulate and detect design clashes between an old bridge and a new bridge. Or a vehicle driving around a narrow street, we can find out if it'll fit the entire way and adjust our plan if it doesn't.

Data Capture

Recording information or data using digital methods.

Using either a computer or mobile device to record data. This data could be a photo, text attached to a data field such as description or location detail, an answer to a yes/no question, a status change or a comment added to a task or checklist.

Digital

A method of working that uses technology or computing.

The term digital ref

Element

An item on site, a column, coper, tile, wall, door, panel...

An "element" could refer to anything on site. Anything that the project team want to track. Usually this would refer to materials or an aspect of the design. This could be something that we are refurbishing or installing.

LIDAR

Light Detection and Ranging

Lidar (also called LIDAR, LiDAR, and LADAR) is a surveying method that measures distance to a target by illuminating that target with a pulsed laser light and measuring the reflected pulses with a sensor. Differences in laser return times and wavelengths can then be used to make digital representations of the target (a point cloud).

Mundane

The routine or repetitive

We refer to any task that can be automated. The most common of all is manually uploading documents or photos, or manually emailing them to your team. Using BIM all this can be done at the touch of a button. If we can reduce the time it takes to complete this task, we’ll be removing the mundane too.

Point Cloud

A digital collection of measurements.

A bit like a work of art that is made up lots of little coloured dots, a point cloud is a collection of little digital points.

The main difference is that each point has been measured to a certain accuracy, which means if we want to we can find any distance.

Its a set of data points in some coordinate system. In a three-dimensional coordinate system, these points are usually defined by X, Y, and Z coordinates, and often are intended to represent the external surface of an object. Point clouds may be created by 3D scanners.

Tools

An aspect of BIM, a bit like a bolt on for your project.

A "BIM" Tool is a feature or aspect of BIM. It’s something that can be added to your project to improve efficiency. This could be a digital photo capture or a 3D Simulation of the works created from the programme.

Value

A positive outcome resulting in time or money saved.

We talk about value, what we're saying here is a positive or better alternative to the previous method of work.
